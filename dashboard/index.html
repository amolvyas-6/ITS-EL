Mathematical Foundation

Graph Model

The metro station is modeled as an undirected weighted graph G = (V, E, w) where:

  • V = set of 16 nodes (physical locations)
  • E = set of 25 edges (walkable paths)
  • w: E → ℝ⁺ = edge weights (distances)

Markov Chain

Crowd movement follows a discrete-time Markov chain with:

  • State = current node of a person
  • T = row-stochastic transition matrix
  • P(t+1) = P(t) · T (evolution equation)
  • π = π · T (stationary distribution)

Optimization

Three strategies applied to reduce congestion:

  • Exit Acceleration — Stronger bias toward exits
  • Bottleneck Relief — Redistribute flow away from congested nodes
  • Path Splitting — Balance load across parallel corridors
